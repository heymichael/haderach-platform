This page documents the current test model for haderach and where test status will be published.

Current Test Model

Haderach testing follows the same three-suite model used by app docs surfaces:

  • Deploy Smoke: blocking subset for deploy-linked changes.
  • Nightly E2E Regression: full non-blocking regression run on schedule.
  • Prod Monitor: non-blocking production canary checks.
